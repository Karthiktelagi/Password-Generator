# Password Generator Web App

A simple web application that generates secure passwords based on user-defined length.  
Built using Python (Flask) for the backend and HTML, CSS, and JavaScript for the frontend.

## Features
- User-defined password length
- Secure password generation logic
- Backend input validation
- Clean and minimal user interface

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Version Control: Git, GitHub

## How to Run

### Prerequisites
- Python 3.8 or higher
- pip (Python package manager)

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/Karthiktelagi/Password-Generator.git
2. Go to project folder → `cd Password-Generator`  
3. Create virtual environment → `python -m venv venv`  
4. Activate (Windows) → `venv\Scripts\activate`  
5. Install Flask → `pip install flask`  
6. Run app → `python app.py`  
7. Open browser → `http://127.0.0.1:5000`



## Description
This project demonstrates basic full-stack web development using Python and Flask.
